Profile of Ruiqi

Project Description:
Profile of Ruiqi is an interactive portfolio website built with Flask, HTML, CSS, JavaScript, and the OpenAI API. Instead of presenting my resume as a traditional static webpage, this project turns my portfolio into a fictional “Harvard Bureau of Investigation” agent archive. Visitors begin on a search page where the agent name is locked as Ruiqi Lai. After clicking search, they enter an agent profile page that introduces different “identity files” connected to my work as a designer, researcher, artist, and hidden identity.

The goal of the project is to make a personal portfolio feel more interactive, memorable, and narrative-driven. Each identity page presents a different side of my background. The Designer page shows selected design projects and demo videos. The Researcher page shows research-related projects with checkbox filters. The Artist page links to my older art portfolio. The Hidden Identity page is password-protected and reveals a playful personal page. The site also includes an AI Ruiqi chatbot that uses the OpenAI API to help visitors decide which project or identity page they should explore.


Features:
- A fictional HBI search interface as the landing page.
- An agent profile page with avatar images and identity cards.
- Hover-based identity cards that reveal avatar images.
- A Designer identity page with project descriptions, tags, and demo videos.
- A Researcher identity page with checkbox filters for project categories.
- An Artist identity page with a link to my original portfolio.
- A password-protected Hidden Identity page.
- An AI Ruiqi chat page powered by the OpenAI API.
- Chat responses that can generate clickable page buttons such as “Open Designer Page” or “Open Researcher Page.”
- A single-page navigation structure where different pages are shown and hidden using JavaScript.
- Responsive layouts and custom CSS styling using a black-and-white file/archive visual style.
